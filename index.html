Weather App

Introducing my weather application! This app fetches weather data for any city or the user's current location using a weather API. It dynamically displays the temperature, condition, humidity, wind speed, and date, and allows the user to toggle between Celsius and Fahrenheit units. The user interface is built with HTML, CSS, and Bootstrap, while the app's functionality is powered by JavaScript. It's a great example of how these technologies can work together to create a practical and user-friendly application. So, if you're looking for an easy way to check the weather, give my app a try! I hope you find it useful.

World Clock

Introducing my world clock application! This app displays the current date and time for selected cities, including options for the current location, London, New York, and Auckland. It uses Moment.js and Moment Timezone to handle time calculations and time zone conversions, and dynamically updates the time every second. The user interface is built with HTML, CSS, and JavaScript, making for a clean and straightforward experience. It's a great example of how these technologies can work together to create a useful and practical application. So, if you're looking for an easy way to keep track of time across different time zones, give my app a try! I hope you find it helpful.
